# Dice Rolling Game 🎲

A simple Python program that simulates rolling two dice.

## Features

* Roll two dice randomly.
* Displays the result of each die.
* Allows the user to roll multiple times.
* Handles invalid input gracefully.

## How It Works

The program asks the user whether they want to roll the dice.

* Enter `y` to roll the dice.
* Enter `n` to quit the game.
* Any other input will display an error message.

Each roll generates two random numbers between 1 and 6, representing two dice.

## Requirements

* Python 3.x


## Example Output

```text
Roll the dice? (y/n): y
(4, 2)

Roll the dice? (y/n): y
(6, 6)

Roll the dice? (y/n): n
Thanks for playing!
```

## Concepts Used

* Python loops (while)
* Conditional statements (if, elif, else)
* User input (input)
* Random number generation (random.randint)
* String methods (lower)
* Loop control using break
