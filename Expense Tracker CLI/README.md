# Expense Tracker CLI
A simple command-line expense tracker built with Python that helps users manage their daily expenses. The application stores all expense data in a JSON file, allowing users to save and access their records across multiple sessions.

---

## Features

- Add a new expense
- View all expenses
- Filter expenses by category
- View spending summary
- Calculate remaining budget
- Calculate daily budget
- Save and load expenses using a JSON file
- Menu-driven CLI

---

## Technologies Used

- Python
- JSON
- datetime module
- calendar module

## How to Run

### Step 1: Clone the Repository
```bash
git clone https://github.com/aaayushie/python-projects-for-beginners.git
```

### Step 2: Navigate to the Project Folder
```bash
cd python-projects-for-beginners
cd "Expense Tracker CLI"
```

### Step 3: Run the Application
```bash
python expense_tracker.py
```

### Step 4: Use the Application
After running the program, you'll see the following menu:
```text
Expense Tracker

1. Add Expense
2. View All Expenses
3. Filter Expenses
4. View Summary
5. Exit
```
Enter the corresponding number to perform the desired operation.
